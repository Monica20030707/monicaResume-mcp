# Experience Entry Template — CUSTOMIZE THIS
Create one file per work experience: company-role.yaml  
The build agent selects the most relevant entries per job posting

## Basic Information
- **Company:** Company Name
- **Title:** Software Engineer
- **Type:** full-time  _(full-time, contract, internship)_
- **Location:** City, State or Remote
- **Start Date:** Jan 2024
- **End Date:** Present
- **Company Description:** Brief description of the company.

## Bullets
- Default resume-ready bullet point with action verb and metrics
- Another achievement-oriented bullet point

## Tech Used
- TypeScript
- React
- PostgreSQL

## Bullet Variants
Bullet variants let the build agent pick the best framing per job

### Technical
- Same achievement reframed to emphasize technical depth
- Another technically-focused bullet

### Impact
- Same achievement reframed to emphasize measurable results
- Another impact-focused bullet

### Leadership
- Same achievement reframed to emphasize leadership and ownership
- Another leadership-focused bullet
